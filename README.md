# Production-Grade RAG AI Assistant

A Retrieval-Augmented Generation (RAG) application built with Python, FastAPI, LlamaIndex, Qdrant, and Streamlit.

link : https://enterprise-rag-system-5mlance6rxp9ejkskoyvxd.streamlit.app/

## Features

- PDF document ingestion
- Semantic search with vector embeddings
- AI-powered question answering
- FastAPI backend
- Qdrant vector database
- Streamlit frontend
- Production-ready architecture

## Tech Stack

- Python
- FastAPI
- LlamaIndex
- Qdrant
- Streamlit

## Installation

```bash
pip install -r requirements.txt
```

## Run Application

```bash
python main.py
```

or

```bash
streamlit run streamlit_app.py
```

## Project Architecture

PDF → Chunking → Embeddings → Qdrant → Retrieval → LLM → Response
